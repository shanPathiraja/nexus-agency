# Nexus Agency — Landing Page

A premium digital agency landing page built with Next.js 14, Tailwind CSS, and Framer Motion.

## Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Fonts:** Cormorant Garamond (display) + DM Sans (body)
- **Deployment:** Vercel

## Sections
- Navigation (sticky, mobile-responsive)
- Hero (animated headline rotator, parallax)
- Work (project grid with hover effects)
- Services (expandable rows)
- Process (alternating timeline)
- Testimonials (client quotes)
- Contact (form + gold CTA)
- Footer

## Deploy to Vercel

1. Upload this folder to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Framework: Next.js (auto-detected)
4. Click Deploy

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000
